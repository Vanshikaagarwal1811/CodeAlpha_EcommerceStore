# CodeAlpha E-Commerce Store

A fully functional E-Commerce website built using Django, HTML, CSS, and JavaScript.

## Features
- User Registration & Login
- Product Listings
- Product Detail Page
- Shopping Cart
- Checkout System
- Order Placement
- Responsive Design
- Animated Empty Cart UI

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
